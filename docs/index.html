为什么是 PenMark

它不是另一套在线协作文档。

它是一张属于个人的长期纸张。富文本友好，本地优先，内容可迁移。

印象笔记、有道云等产品可能存在广告、设备和产品策略限制。Obsidian 本地可靠，但 Markdown 和同步有学习成本。Notion、飞书更偏云端结构化或协作。

PenMark 希望提供一张本地优先、富文本友好的个人纸张——打开就能写，内容永远带得走。

功能亮点

该有的都有，不该挡的不挡

打开就写

双击启动，无需登录、联网或初始化。新建文档目标 < 100ms。

富文本粘贴

公众号、飞书、网页图文粘贴尽量保真，图片自动修复。

图片拖入与缩放

拖入即插入，支持缩放、对齐、复制和排列。

文件夹与大纲

文件夹分类管理，标题自动生成目录大纲。

链接卡片

自动抓取链接元数据，生成可视化卡片。

本地数据

数据保存在本机，不依赖云服务，可随时备份。

多格式导出

支持 Word、HTML、Markdown、长图导出和资料库批量导出。

旧数据导入

支持旧版 PenMark SQLite 数据库安全导入，只新增不覆盖。

快捷键

对齐飞书/在线文档肌肉记忆，Ctrl+/ 查看全部快捷键。

软件可以更换，内容不该被锁住。
