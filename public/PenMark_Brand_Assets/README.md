# 知著 PenMark 品牌资产

## 核心概念
- 纸张轮廓：内容与文档
- 右下折角：完成、导出与流转
- 输入光标：正在创作
- 中文粗字重：稳定、清晰、有识别度
- 英文轻字重：现代、轻量、克制

## 标准颜色
- 暖米白：`#F7F4EC`
- 深炭灰：`#20292E`
- 深色背景：`#182126`
- 暖白文字：`#F3F1EB`
- 鼠尾草绿：`#8BA393`

## 文件使用
- `penmark-icon-*.svg`：独立图标，适合导航栏、头像与小尺寸界面
- `penmark-logo-horizontal-*.svg`：官网页头、登录页、产品介绍
- `penmark-logo-stacked-*.svg`：启动页、宣传页、居中展示
- `penmark-app-icon.svg/png`：PWA、桌面快捷方式、应用图标
- `favicon.svg/ico`：浏览器标签页
- `apple-touch-icon.png`：iOS 主屏幕图标

## 网页示例
```html
<link rel="icon" href="/branding/favicon.svg" type="image/svg+xml">
<link rel="apple-touch-icon" href="/branding/apple-touch-icon.png">

<img src="/branding/penmark-logo-horizontal-light.svg" alt="知著 PenMark">
```

## 安全留白
LOGO 四周至少保留一个“输入光标宽度”的空白距离，不建议压边使用。

## 禁止事项
- 不添加钢笔、毛笔、印章或复杂中式纹样
- 不改变图标折角方向
- 不拉伸、压扁或增加立体渐变
- 不在低对比度背景上直接使用
